Custom Guitars is a colaberative project completed for 2nd year. 
It is an ecommerece website with an extensive ammount of functionality including 2FA compatible with authenticator apps such as Microsoft authenticator, Full product buying system with includes adding to a cart, checking out and payment with stripe, and much more.
A signifact part of our project is the 3D guitar viewer built in ThreeJs. This loaded 3d models and using gyroscopic controls you are able to move it around in 3d. Custom lighting with ambient and directional lights make the different textures on each model visable.
